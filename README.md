## Installation

1. Clone repository
```
git clone https://github.com/golushkin/drf_film.git
```
2. Go to dir drf_film
```
cd drf_film
```
3. Install pipenv
```
pip install pipenv
```
4. Install dependencies
```
pipenv install
```
5. Run server
```
pipenv shell
py manage.py runserver
```
